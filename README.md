# TaskManager API

Educational ASP.NET Core Web API project.

## Features

- ASP.NET Core Controllers
- DTOs
- Dependency Injection
- Middleware
- Exception Handling
- EF Core + SQLite
- Repository Pattern
- Async CRUD operations

## Technologies

- C#
- ASP.NET Core
- EF Core
- SQLite

## Purpose

Learning backend architecture and request pipeline basics.
